# Di-Battista-et-al-Impact-of-Repetitive-Exposure-to-Low-level-Blast-on-Neurocognitive-Function-in-CAF

Code used in the manuscript "The Impact of Repetitive Exposure to Low-level Blast on Neurocognitive Function in Canadian Armed Forces' Breachers, Snipers and Military Controls" By Alex P. Di Battista, Shawn G. Rhind, Catherine Tenn, Ann Nakashima, Timothy K. Lam, Maria Y. Shiu, Kristen King, and Oshin Vartanian.

Nota bene: The data belong to the Canadian Armed Forces, and hence simulated data are provided to run the code and recreate the models, tables, and figures used in the manuscript. No data are provided for table creation due to security reasons; however, the code for table creation is provided.
